# Как получить токен доступа GitVerse

Этот документ объясняет, куда лезть и что нажимать, чтобы получить токен доступа для Macaroni Messenger на примере GitVerse.

Важно: GitVerse browser adapter в Macaroni Messenger пока не реализован. Эта инструкция фиксирует будущий auth flow и пригодится, когда adapter появится. Сейчас для реального remote sync используйте GitHub.

Почему GitVerse?

Потому что в таком случае оператором ваших персональных данных станет Сбербанк, а это смешно.

Официальная документация GitVerse: [Токены](https://gitverse.ru/docs/collaborative/authentification/tokens/).

## Что такое токен

Токен доступа заменяет пароль при работе с git-репозиторием и API.

Macaroni Messenger использует токен, чтобы читать и записывать файлы сообщений в выбранный репозиторий.

Важно:

- токен даёт доступ к вашему репозиторию;
- токен нельзя выкладывать в публичный чат;
- токен нельзя коммитить в репозиторий;
- Macaroni Messenger хранит токен в `localStorage`;
- `localStorage` - это удобно, но не банковский сейф.

## Как создать токен

1. Откройте [GitVerse](https://gitverse.ru/).
2. Войдите в аккаунт.
3. Нажмите на иконку пользователя в правом верхнем углу.
4. Откройте **Настройки**.
5. Перейдите во вкладку **Управление токенами**.
6. Введите понятное имя токена, например `macaroni-messenger`.
7. Отметьте чекбокс **Репозитории**.
8. Нажмите **Генерировать токен**.
9. Скопируйте и сохраните токен сразу после генерации.
10. Вставьте токен в поле "Токен доступа" в Macaroni Messenger.

Важно: GitVerse показывает токен только один раз. После перезагрузки страницы он больше не отобразится.

## Какие доступы нужны

Для MVP нужен минимальный набор прав, который позволяет:

- читать файлы репозитория;
- создавать или обновлять файлы сообщений;
- отправлять изменения в репозиторий.

В официальной инструкции GitVerse для работы с репозиториями нужно отметить чекбокс **Репозитории**. Не выдавайте токену доступ шире, чем нужно для переписки через конкретный репозиторий.

## Что делать после создания

1. Создайте или выберите репозиторий для сообщений.
2. Убедитесь, что репозиторий доступен вашему аккаунту.
3. Скопируйте URL репозитория.
4. Когда GitVerse adapter будет реализован, в Macaroni Messenger выберите provider `GitVerse`.
5. Вставьте URL репозитория.
6. Вставьте токен.
7. Сохраните настройки.

## Если токен утёк

1. Откройте настройки токенов GitVerse.
2. Отзовите старый токен.
3. Создайте новый токен.
4. Обновите токен в Macaroni Messenger.

Git помнит.

Интернет помнит.

Но токен хотя бы можно отозвать.
